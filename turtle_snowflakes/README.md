# Turtle Snowflakes (Снежинки с Turtle)

Простой проект на Python — программа для рисования случайных снежинок с помощью библиотеки `turtle`.  
Каждая снежинка имеет случайный размер, цвет и поворот.

## Как пользоваться

1. Запустите `turtle_snowflakes.py`.
2. Введите количество снежинок, которое хотите нарисовать.
3. Наблюдайте, как снежинки появляются в случайных местах с разными цветами и размерами.
4. Закройте окно turtle, когда закончите.

## Цель проекта

- Практика работы с Python: функции, циклы, случайные числа, графика через turtle.  
- Изучение работы с цветами в формате RGB.  
- Развитие логического мышления и навыков визуализации через программирование.
